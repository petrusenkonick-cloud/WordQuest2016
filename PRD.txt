🔄 Главный флоу
Шаг 1: Регистрация
Родитель:
1. Заходит на сайт
2. Регистрируется (email/Google)
3. Создаёт профиль ребёнка (имя, возраст, класс)
4. Получает код для ребёнка
5. Подключает Telegram бота

Ребёнок:
1. Заходит на сайт
2. Вводит код от родителя
3. Выбирает аватар
4. Готов играть

🛡 Риски и митигация
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  РИСКИ:                                                     │
│                                                             │
│  ⚠️ AI неправильно распознаёт домашку                       │
│     → Кнопка "Не то распознал" + ручной ввод                │
│     → Улучшение промптов                                    │
│     → Фидбек loop для обучения                              │
│                                                             │
│  ⚠️ AI даёт неправильные объяснения                         │
│     → Проверка ответов по базе знаний                       │
│     → Родитель может пожаловаться                           │
│     → Модерация контента                                    │
│                                                             │
│  ⚠️ Дети не хотят использовать                              │
│     → Больше геймификации                                   │
│     → Реальные награды (родитель платит за достижения)      │
│     → Социальные фичи (друзья, соревнования)                │
│                                                             │
│  ⚠️ Родители не доверяют AI                                 │
│     → Прозрачность (показываем что AI делает)               │
│     → Родитель может проверить ответы                       │
│     → Отзывы и кейсы                                        │
│                                                             │
│  ⚠️ Конкуренты                                              │
│     → Быстрый запуск                                        │
│     → Фокус на UX и геймификации                            │
│     → Niche down (Ontario curriculum сначала)               │
│                                                             │
│  ⚠️ Стоимость AI вырастет                                   │
│     → Кэширование частых запросов                           │
│     → Оптимизация промптов                                  │
│     → План B: другие модели                                 │
│
